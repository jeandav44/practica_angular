Description of the application


ER

Este practica constara de cuatro tablas, una de usuarios, otra de artículos, otra de categorías y otra de cartas, cada una con sus respectivos campos. El usuario podrá crear usuarios, artículos, categorías y cartas. También podrá visualizar cada una de las tablas y podrá modificar y/o eliminar usuarios, artículos, categorías o cartas.
